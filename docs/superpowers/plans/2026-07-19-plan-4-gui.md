# GUI Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the minimal-but-real Tauri v2 + xterm.js hub GUI (Plan 4): a desktop app whose Rust backend connects to the daemon over `~/.hub/hubd.sock` using the frozen `hub-proto`/`hub-transport` contract types and exposes Tauri commands + events, and whose Svelte frontend renders a tiled grid of live terminals, a reconciliation-bucketed session list, focus-follows-size drag resizing, Hub-origin spawn, and a scrollback-buffer setting.

**Architecture:** The webview (Svelte + xterm.js) never touches the socket. It calls Tauri **commands** (`invoke`) to send control/data frames and subscribes to Tauri **events** for streamed output. A single background actor task in the Rust backend owns one `FramedConn` to the daemon, multiplexes all sessions by `session_id`, writes command frames from an mpsc queue, and reads frames in the same `select!` loop — dispatching `Replay`/`Data`/`Closed`/`Error` to the webview as events and fulfilling the pending `List` reply via a oneshot. Reconciliation buckets (healthy/ghost/orphan) are computed in the backend by diffing the daemon's live `Sessions` against `~/.hub/sessions/<id>.json` record files, because the protocol exposes no bucket message.

**Tech Stack:** Tauri v2 (Rust backend + OS webview), `hub-proto` + `hub-transport` (path deps into the cargo workspace), tokio, serde/serde_json; Svelte 5 + Vite + TypeScript, `@xterm/xterm` + `@xterm/addon-fit`; Rust `#[tokio::test]` unit tests with a mock daemon over a temp unix socket; Playwright smoke tests against a mocked-IPC Vite build.

**Why Svelte (not vanilla TS):** Svelte compiles components to small imperative DOM updates with **no virtual-DOM runtime**, so the shipped bundle stays tiny — directly serving the spec's "lightweight footprint + nice GUI" goal (§2, §16). Its `writable` stores give first-class reactivity for the constantly-changing session/bucket lists, and it is the framework Tauri v2's `create-tauri-app` scaffolds cleanly on top of Vite. Vanilla TS would mean hand-rolling all of that.

Append to `daemon.rs`:

```rust
pub struct DaemonClient {
    /// Pre-encoded frames queued for the writer side of the actor.
    cmd_tx: mpsc::Sender<Vec<u8>>,
    /// Fulfilled by the actor when the next `Sessions` frame arrives.
    pending_list: Arc<Mutex<Option<oneshot::Sender<Vec<SessionInfo>>>>>,
}

impl DaemonClient {
    pub async fn connect(sock: &Path, sink: Arc<dyn EventSink>) -> anyhow::Result<DaemonClient> {
        let conn = connect(sock).await?;
        let (cmd_tx, cmd_rx) = mpsc::channel::<Vec<u8>>(256);
        let pending_list = Arc::new(Mutex::new(None));
        tokio::spawn(actor_loop(conn, cmd_rx, pending_list.clone(), sink));
        Ok(DaemonClient { cmd_tx, pending_list })
    }

    /// Queue a pre-encoded control/data frame for writing.
    pub async fn send(&self, frame_bytes: Vec<u8>) -> anyhow::Result<()> {
        self.cmd_tx
            .send(frame_bytes)
            .await
            .map_err(|_| anyhow::anyhow!("daemon actor stopped"))
    }

    /// Send `List` and await the next `Sessions` reply.
    pub async fn list(&self) -> anyhow::Result<Vec<SessionInfo>> {
        let (tx, rx) = oneshot::channel();
        *self.pending_list.lock().await = Some(tx);
        self.send(encode_control(&ControlMsg::List)).await?;
        Ok(rx.await.map_err(|_| anyhow::anyhow!("list reply dropped"))?)
    }
}

async fn actor_loop(
    mut conn: FramedConn,
    mut cmd_rx: mpsc::Receiver<Vec<u8>>,
    pending_list: Arc<Mutex<Option<oneshot::Sender<Vec<SessionInfo>>>>>,
    sink: Arc<dyn EventSink>,
) {
    loop {
        tokio::select! {
            biased;
            // Writes drain first so keystrokes are never starved by a read.
            maybe = cmd_rx.recv() => match maybe {
                Some(bytes) => { if conn.write_frame(&bytes).await.is_err() { break; } }
                None => break, // all DaemonClient handles dropped
            },
            frame = conn.read_frame() => match frame {
                Ok(Frame::Control(ControlMsg::Sessions { sessions })) => {
                    if let Some(tx) = pending_list.lock().await.take() { let _ = tx.send(sessions); }
                }
                Ok(Frame::Control(ControlMsg::Replay { id, screen })) => {
                    sink.emit_json("hub://replay",
                        serde_json::json!({ "id": id.0, "bytes": screen }));
                }
                Ok(Frame::Control(ControlMsg::Closed { id, exit_code })) => {
                    sink.emit_json("hub://closed",
                        serde_json::json!({ "id": id.0, "exitCode": exit_code }));
                }
                Ok(Frame::Control(ControlMsg::Error { message })) => {
                    sink.emit_json("hub://error", serde_json::json!({ "message": message }));
                }
                Ok(Frame::Data { id, bytes }) => {
                    sink.emit_json("hub://output",
                        serde_json::json!({ "id": id.0, "bytes": bytes }));
                }
                Ok(_) => { /* Open/Opened/Attach/etc. are not inbound to the hub; ignore */ }
                Err(_) => break, // connection closed
            },
        }
    }
}
```

Note: `tokio::AsyncRead::read` (used inside `FramedConn::read_frame`) is cancellation-safe and the decoder buffers partial reads, so cancelling the read arm of `select!` to service a write loses no bytes. ⚠️ executor: this relies on `FramedConn::read_frame` reading into its internal `FrameDecoder` before returning (contract-conformant); confirm when Plan 1 lands.

use std::collections::HashSet;
use hub_proto::{Origin, SessionInfo};
use serde::{Deserialize, Serialize};

/// On-disk `~/.hub/sessions/<id>.json` shape: SessionInfo fields (flattened) + sock + record_version.
#[derive(Clone, Debug, Serialize, Deserialize)]
pub struct GhostRecord {
    pub id: u64,
    pub origin: Origin,
    pub title: String,
    pub pid: u32,
    pub started_unix: u64,
    pub cols: u16,
    pub rows: u16,
    pub sock: String,
    pub record_version: u32,
}

#[derive(Serialize)]
pub struct Buckets {
    pub healthy: Vec<SessionInfo>,
    pub ghost: Vec<GhostRecord>,
    pub orphan: Vec<SessionInfo>,
}

pub fn bucketize(live: Vec<SessionInfo>, records: Vec<GhostRecord>) -> Buckets {
    let live_ids: HashSet<u64> = live.iter().map(|s| s.id.0).collect();
    let record_ids: HashSet<u64> = records.iter().map(|r| r.id).collect();

    let healthy = live.iter().filter(|s| record_ids.contains(&s.id.0)).cloned().collect();
    let orphan = live.iter().filter(|s| !record_ids.contains(&s.id.0)).cloned().collect();
    let ghost = records.into_iter().filter(|r| !live_ids.contains(&r.id)).collect();
    Buckets { healthy, ghost, orphan }
}

// Deterministic in-memory IPC used only when VITE_MOCK is set.
// send_input(id, bytes) echoes the same bytes back as a hub://output event,
// so typing into an xterm tile visibly echoes without a real backend.
type Handler = (e: { payload: any }) => void;
const handlers: Record<string, Handler[]> = {};

function emit(event: string, payload: any) {
  (handlers[event] ?? []).forEach((h) => h({ payload }));
}

export async function invoke<T>(cmd: string, args?: any): Promise<T> {
  switch (cmd) {
    case "list_sessions":
      return [{ id: 1, origin: "Hub", title: "mock", pid: 0, started_unix: 0, cols: 80, rows: 24 }] as unknown as T;
    case "reconcile_sessions":
      return {
        healthy: [{ id: 1, origin: "Hub", title: "mock", pid: 0, started_unix: 0, cols: 80, rows: 24 }],
        ghost: [],
        orphan: [],
      } as unknown as T;
    case "attach":
      // Simulate REPLAY on attach.
      setTimeout(() => emit("hub://replay", { id: args.id, bytes: Array.from(new TextEncoder().encode("$ ")) }), 5);
      return undefined as unknown as T;
    case "send_input":
      // Echo keystrokes back as live output.
      setTimeout(() => emit("hub://output", { id: args.id, bytes: args.bytes }), 1);
      return undefined as unknown as T;
    case "get_buffer_size":
      return 10000 as unknown as T;
    default:
      return undefined as unknown as T;
  }
}

export async function listen(event: string, handler: Handler) {
  (handlers[event] ??= []).push(handler);
  return () => {
    handlers[event] = (handlers[event] ?? []).filter((h) => h !== handler);
  };
}

```svelte
<script lang="ts">
  import { onMount, onDestroy } from "svelte";
  import { Terminal } from "@xterm/xterm";
  import { FitAddon } from "@xterm/addon-fit";
  import "@xterm/xterm/css/xterm.css";
  import { listen, attach, detach, sendInput, claimSize, type UnlistenFn } from "./api";

  let { sessionId, scrollback = 10000 }: { sessionId: number; scrollback?: number } = $props();

  let el: HTMLDivElement;
  let term: Terminal;
  let fit: FitAddon;
  let unlisten: UnlistenFn[] = [];
  let resizeTimer: number | undefined;
  let lastCols = 0;
  let lastRows = 0;
  let ro: ResizeObserver;

  function claimNow() {
    fit.fit();
    const dims = fit.proposeDimensions();
    if (!dims) return;
    // "only resize if dims changed" (spec §7) to avoid reflow thrash.
    if (dims.cols === lastCols && dims.rows === lastRows) return;
    lastCols = dims.cols;
    lastRows = dims.rows;
    claimSize(sessionId, dims.cols, dims.rows);
  }
  function claimSoon() {
    if (resizeTimer) clearTimeout(resizeTimer);
    resizeTimer = window.setTimeout(claimNow, 50); // ~50ms debounce (spec §7)
  }

  onMount(async () => {
    term = new Terminal({ scrollback, convertEol: false, cursorBlink: true, fontSize: 13 });
    fit = new FitAddon();
    term.loadAddon(fit);
    term.open(el);
    fit.fit();

    // Keystrokes -> daemon as a Data frame (INPUT). (Contract: input is Frame::Data.)
    term.onData((data: string) => {
      sendInput(sessionId, Array.from(new TextEncoder().encode(data)));
    });

    // Focus-follows-size: gaining focus on this tile claims pty sizing.
    term.textarea?.addEventListener("focus", claimNow);

    // Register REPLAY + OUTPUT listeners BEFORE attach so no frame is missed.
    unlisten.push(
      await listen<{ id: number; bytes: number[] }>("hub://replay", (e) => {
        if (e.payload.id !== sessionId) return;
        term.write(new Uint8Array(e.payload.bytes)); // REPLAY snapshot first
      }),
    );
    unlisten.push(
      await listen<{ id: number; bytes: number[] }>("hub://output", (e) => {
        if (e.payload.id !== sessionId) return;
        term.write(new Uint8Array(e.payload.bytes)); // then live stream, in order
      }),
    );

    // Drag-resize of the wrapping tile drives claim_size (debounced).
    ro = new ResizeObserver(() => claimSoon());
    ro.observe(el);

    // Attach LAST — this triggers the relay's Replay + SIGWINCH nudge + live stream.
    await attach(sessionId);
  });

  onDestroy(async () => {
    ro?.disconnect();
    unlisten.forEach((u) => u());
    // Detach = stop viewing; the session keeps running (distinct from kill).
    try { await detach(sessionId); } catch {}
    term?.dispose();
  });
</script>

<div class="tile" bind:this={el} tabindex="0"></div>

<style>
  .tile { width: 100%; height: 100%; background: #000; }
</style>

The SIGWINCH refresh nudge from the relay (spec §7, §10) arrives as ordinary `Data` and repaints full-screen TUIs.

use std::path::Path;
use serde::{Deserialize, Serialize};

pub const DEFAULT_SCROLLBACK: u32 = 10_000;

#[derive(Serialize, Deserialize)]
struct Config { scrollback: u32 }

fn config_path(dir: &Path) -> std::path::PathBuf { dir.join("config.json") }

pub fn load_scrollback(dir: &Path) -> u32 {
    match std::fs::read_to_string(config_path(dir)) {
        Ok(txt) => serde_json::from_str::<Config>(&txt).map(|c| c.scrollback).unwrap_or(DEFAULT_SCROLLBACK),
        Err(_) => DEFAULT_SCROLLBACK,
    }
}

pub fn store_scrollback(dir: &Path, size: u32) -> anyhow::Result<()> {
    std::fs::create_dir_all(dir)?;
    let txt = serde_json::to_string_pretty(&Config { scrollback: size })?;
    std::fs::write(config_path(dir), txt)?;
    Ok(())
}

#[tauri::command]
pub fn get_buffer_size() -> u32 {
    load_scrollback(&crate::hub_home())
}

#[tauri::command]
pub fn set_buffer_size(size: u32) -> Result<(), String> {
    store_scrollback(&crate::hub_home(), size).map_err(|e| e.to_string())
}
```
